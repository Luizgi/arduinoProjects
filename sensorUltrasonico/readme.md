# Medição de Distância com Sensor Ultrassônico e Arduino

Neste projeto, você aprenderá a medir a distância usando um Sensor Ultrassônico (HC-SR04) e um Arduino. Vamos explicar a configuração física e o código passo a passo.

## Configuração Física

### Materiais Necessários

- Arduino Uno
- Sensor Ultrassônico HC-SR04
- Jumpers

### Instruções

1. Conecte o Sensor Ultrassônico:
   - Conecte o pino Trigger do Sensor Ultrassônico ao pino 4 do Arduino.
   - Conecte o pino Echo do Sensor Ultrassônico ao pino 5 do Arduino.
   - Alimente o Sensor Ultrassônico com +5V (5 volts) e conecte o terra (GND) do Sensor Ultrassônico ao terra (GND) do Arduino.

2. Conecte o Arduino a um computador via USB para alimentação e comunicação.
